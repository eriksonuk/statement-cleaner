Statement Cleaner

Перетащите выгрузку Payment Transactions из BO — инструмент уберёт служебные и чувствительные поля, исправит суммы (копейки → рубли/доллары) и оставит только депозиты и выводы. Файл обрабатывается прямо в браузере и никуда не отправляется.

Что делает инструмент: оставляет колонки Date, Type, Payment Method, Amount (RUB), Amount (USD), Status. Удаляет ID транзакций, Provider External ID, Exchange Rate, комиссии, баланс, бонус-коды и поле Notes (номера карт, телефон, гео). Суммы: RUB = PS. Value ÷ 100, USD = Value ÷ 100. Строки, кроме Deposit/Withdraw, отбрасываются. В отчёт попадают только успешные транзакции (Processed / Success / Completed); pending, declined и failed отрезаются.
